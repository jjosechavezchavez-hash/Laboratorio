🔗 Conexión con Google Sheets
Para guardar los resultados automáticamente en Google Sheets, necesitas crear un Google Apps Script que reciba los datos.

Pasos:
1. Ve a script.google.com y crea un nuevo proyecto.
2. Pega el código que te mostramos abajo en el editor.
3. Haz clic en "Implementar → Nueva implementación → Aplicación web".
4. En "Quién tiene acceso" elige Cualquier persona.
5. Copia la URL de la implementación y pégala aquí abajo.

Código para Google Apps Script
function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet()
              .getActiveSheet();
  var data = JSON.parse(e.postData.contents);

  // Encabezados (solo primera vez)
  if (sheet.getLastRow() === 0) {
    sheet.appendRow([
      'Fecha', 'Hora', 'Nombres', 'Apellidos',
      'Jornada', 'No. Máquina',
      'Comp. Correctas', 'Comp. Puntos',
      'Prog. Correctas', 'Prog. Puntos',
      'Total'
    ]);
  }

  var end = new Date(data.endTime);
  sheet.appendRow([
    end.toLocaleDateString('es-GT'),
    end.toLocaleTimeString('es-GT'),
    data.nombres, data.apellidos,
    data.jornada, data.maquina,
    data.compCorrect, data.compPts,
    data.progCorrect, data.progPts,
    data.total
  ]);

  return ContentService
    .createTextOutput(JSON.stringify({status:'ok'}))
    .setMimeType(ContentService.MimeType.JSON);
}
